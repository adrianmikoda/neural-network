# Contribution Guidelines

## Workflow
1. We use [GitHub Flow](https://docs.github.com/en/get-started/using-github/github-flow) (branch off `main`, PR, and squash merge back to `main`).
2. Issues and PR discussions can be in Polish or English, but all code, comments, branch names, and commit messages **must be in English**.

## Commit Message Convention
We use [Conventional Commits](https://www.conventionalcommits.org/).
Allowed scopes: `lib`, `cli`, `gui`.
Examples:
- `feat(lib): add matrix multiplication`
- `fix(cli): resolve dataset parsing error`
- `chore: update github actions`
